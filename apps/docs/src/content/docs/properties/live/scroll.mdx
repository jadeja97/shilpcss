# Scroll Behaviour

Control scroll behavior and other properties.

Usage:

- `@live scroll-*;`
- `@live scroll-auto;`
- `@live scroll-smooth;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    live: {
      scroll: {
        DEFAULT: {
          property: "scroll-behavior: <v><i>;",
          values: {
            auto: "auto",
            smooth: "smooth",
          },
        },
      },
    },
  },
};

export default shilpConfig;
```
