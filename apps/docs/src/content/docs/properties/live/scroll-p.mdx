# Scroll Padding

Set spacing inside element (padding) for snapping or scrolling.

Usage:

- `@live scroll-p-*;`
- `@live scroll-pt-*;`
- `@live scroll-pb-*;`
- `@live scroll-pl-*;`
- `@live scroll-pr-*;`
- `@live scroll-px-*;`
- `@live scroll-py-*;`
- `@live scroll-pbl-*;`
- `@live scroll-pbls-*;`
- `@live scroll-pble-*;`
- `@live scroll-pi-*;`
- `@live scroll-pis-*;`
- `@live scroll-pie-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.
