# Scroll Margin

Set spacing around element (margin) for snapping or scrolling.

Usage:

- `@live scroll-m-*;`
- `@live scroll-mt-*;`
- `@live scroll-mb-*;`
- `@live scroll-ml-*;`
- `@live scroll-mr-*;`
- `@live scroll-mx-*;`
- `@live scroll-my-*;`
- `@live scroll-mbl-*;`
- `@live scroll-mbls-*;`
- `@live scroll-mble-*;`
- `@live scroll-mi-*;`
- `@live scroll-mis-*;`
- `@live scroll-mie-*;`
