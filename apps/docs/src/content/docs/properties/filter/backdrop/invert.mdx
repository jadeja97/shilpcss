# Backdrop Invert Filter

Invert backdrop colors with filter effect.

Usage:

- `@filter backdrop-invert-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    filter: {
      backdrop: {
        invert: {
          property: "backdrop-filter: invert(<v>)<i>;",
          values: {
            DEFAULT: 1,
            none: 0,
          },
        },
      },
    },
  },
};

export default shilpConfig;
```
