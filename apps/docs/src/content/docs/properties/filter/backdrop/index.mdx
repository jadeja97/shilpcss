# Backdrop Filter

Apply backdrop filter effects to elements.

Usage:

- `@filter backdrop-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    filter: {
      backdrop: {
        DEFAULT: {
          property: "backdrop-filter: <v><i>;",
          values: {
            // backdrop-filter: [blur] [brightness] [contrast] [drop-shadow] [grayscale] [hue-rotate] [invert] [opacity] [saturate] [sepia];
          },
        },
      },
    },
  },
};

export default shilpConfig;
```
