# Size

Set elements size (width & height) including block and inline.

Usage:

- `@size is-*;`
- `@size is-block-*;`
- `@size is-inline-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    size: {
      is: {
        DEFAULT: {
          property: `
						width: <v><i>;
						height: <v><i>;
					`,
          resolve: "spacing",
          themeKey: "spacing",
          values: {
            min: "min-content",
            max: "max-content",
            fit: "fit-content",
          },
        },

        block: {
          property: "block-size: <v><i>;",
          resolve: "spacing",
          themeKey: "spacing",
          values: {
            min: "min-content",
            max: "max-content",
            fit: "fit-content",
          },
        },

        inline: {
          property: "inline-size: <v><i>;",
          resolve: "spacing",
          themeKey: "spacing",
          values: {
            min: "min-content",
            max: "max-content",
            fit: "fit-content",
          },
        },
      },
    },
  },
};

export default shilpConfig;
```
