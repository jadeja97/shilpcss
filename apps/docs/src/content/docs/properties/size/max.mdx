# Maximum Size

Set maximum dimensions for elements including block and inline.

Usage:

- `@size max-*;`
- `@size max-w-*;`
- `@size max-h-*;`
- `@size max-block-*;`
- `@size max-inline-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>
