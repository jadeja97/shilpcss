# Flex Flow OR Direction

Define flex layout flow or direction.

Usage:

- `@flex flow-*;`
- `@flex flow-row;`
- `@flex flow-row-reverse;`
- `@flex flow-col;`
- `@flex flow-col-reverse;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    flex: {
      flow: {
        property: "flex-direction: <v><i>;",
        values: {
          row: {
            DEFAULT: "row",
            reverse: "row-reverse",
          },
          col: {
            DEFAULT: "column",
            reverse: "column-reverse",
          },
        },
      },
    },
  },
};

export default shilpConfig;
```
