# Margin

Apply spacing around elements (margin).

Usage:

- `@space m-*;`
- `@space mt-*;`
- `@space mb-*;`
- `@space ml-*;`
- `@space mr-*;`
- `@space mx-*;`
- `@space my-*;`
- `@space mbl-*;`
- `@space mbls-*;`
- `@space mble-*;`
- `@space mi-*;`
- `@space mis-*;`
- `@space mie-*;`
