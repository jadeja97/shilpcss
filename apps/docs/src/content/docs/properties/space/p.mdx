# Padding

Apply spacing inside elements (padding).

Usage:

- `@space p-*;`
- `@space pt-*;`
- `@space pb-*;`
- `@space pl-*;`
- `@space pr-*;`
- `@space px-*;`
- `@space py-*;`
- `@space pbl-*;`
- `@space pbls-*;`
- `@space pble-*;`
- `@space pi-*;`
- `@space pis-*;`
- `@space pie-*;`

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.
