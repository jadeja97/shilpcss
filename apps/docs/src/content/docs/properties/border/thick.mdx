# Border Thickness (Width)

Control border width or thickness for styling.

Usage:

- `@border thick-*;`
- `@border thick-top-*;`
- `@border thick-right-*;`
- `@border thick-bottom-*;`
- `@border thick-left-*;`
- `@border thick-x-*;`
- `@border thick-y-*;`
- `@border thick-bl-*;`
- `@border thick-bls-*;`
- `@border thick-ble-*;`
- `@border thick-bi-*;`
- `@border thick-bis-*;`
- `@border thick-bie-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.
