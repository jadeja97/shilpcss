# Border Color

Apply border colors to element borders.

Usage:

- `@border color-*;`
- `@border color-top-*;`
- `@border color-right-*;`
- `@border color-bottom-*;`
- `@border color-left-*;`
- `@border color-x-*;`
- `@border color-y-*;`
- `@border color-bl-*;`
- `@border color-bls-*;`
- `@border color-ble-*;`
- `@border color-bi-*;`
- `@border color-bis-*;`
- `@border color-bie-*;`

This page contains only the **property configuration reference**. Complete documentation and usage
