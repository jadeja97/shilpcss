# Border Style

Define border styles like solid, dashed, dotted and more.

Usage:

- `@border style-*;`
- `@border style-top-*;`
- `@border style-right-*;`
- `@border style-bottom-*;`
- `@border style-left-*;`
- `@border style-x-*;`
- `@border style-y-*;`
- `@border style-bl-*;`
- `@border style-bls-*;`
- `@border style-ble-*;`
- `@border style-bi-*;`
- `@border style-bis-*;`
- `@border style-bie-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.
