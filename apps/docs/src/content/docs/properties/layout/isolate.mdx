# Stacking Context Isolation

Create new stacking context for elements.

Usage:

- `@layout isolate-*;`
- `@layout isolate-auto;`
- `@layout isolate-forced;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    layout: {
      isolate: {
        property: "isolation: <v><i>;",
        values: {

          forced: "isolate",
        },
      },
    },
  },
};
```
