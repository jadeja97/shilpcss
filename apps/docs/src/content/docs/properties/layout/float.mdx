# Float Items

Positions element (float) and letting content wrap around.

Usage:

- `@layout float-*;`
- `@layout float-clear-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

<section>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    layout: {
      float: {
        DEFAULT: {
          property: "float: <v><i>;",
          values: {
            left: "left",
            right: "right",
            none: "none",
          },
        },

        clear: {
          property: "clear: <v><i>;",
          values: {
            left: "left",
            right: "right",
            none: "none",
            both: "both",
          },
        },
      },
    },
  },
};

export default shilpConfig;
```
