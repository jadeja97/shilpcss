# Box Sizing

Control box-sizing model for elements.

Usage:

- `@layout box-*;`
- `@layout box-border;`
- `@layout box-content;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    layout: {
      box: {
        property: "box-sizing: <v><i>;",
        values: {
          border: "border-box",
          content: "content-box",
        },
      },
    },
  },
};

export default shilpConfig;
```
