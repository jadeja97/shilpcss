# Gradient Color Mask

Apply gradient as a mask to elements.

Usage:

- `@mask gradient-*;`
- `@mask gradient-linear-*;`
- `@mask gradient-radial-*;`
- `@mask gradient-conic-*;`

<Alert variant="danger">

Gradient color mask uses `mask-image` property.

So, if you are using `@mask img-*;` then conflict will be created. You need to choose a stratagey to
work with both.

</Alert>

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>
