# Exit Animation

Define exit animations for elements.

Usage:

- `@animate exit;`
- `@animate exit-unset;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    animate: {
      exit: {
        property: "<v>",
        special: true,
        values: {
          DEFAULT: `
						animation-name: exit<i>;
						animation-duration: 0.25s<i>;
					`,
          unset: `
						--ext-opct: unset<i>;
						--ext-scl: unset<i>;
						--ext-rtt: unset<i>;
						--ext-mv-x: unset<i>;
						--ext-mv-y: unset<i>;
					`,
        },
      },
    },
  },
};

export default shilpConfig;
```
