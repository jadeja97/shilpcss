# Animation Name

Apply named animations by specifying keyframe names.

Usage:

- `@animate name-*;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    animate: {
      name: {
        property: "animation-name: <v><i>;",
        values: {
          enter: {
            DEFAULT: "enter",
            exit: "enter, exit",
          },
          exit: "exit",
        },
      },
    },
  },
};

export default shilpConfig;
```
