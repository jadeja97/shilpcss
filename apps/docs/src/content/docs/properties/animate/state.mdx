# Animation Play State

Control animation state transitions and pauses.

Usage:

- `@animate state-*;`
- `@animate state-paused;`
- `@animate state-playing;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    animate: {
      state: {
        property: "animation-play-state: <v><i>;",
        values: {
          paused: "paused",
          playing: "running",
        },
      },
    },
  },
};

export default shilpConfig;
```
