# Entry Animation

Define entry animations for elements appearing on screen.

Usage:

- `@animate enter;`
- `@animate enter-unset;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    animate: {
      enter: {
        property: "<v>",
        special: true,
        values: {
          DEFAULT: `
            animation-name: enter<i>;
            animation-duration: 0.25s<i>;
          `,
          unset: `
            --entr-opct: unset<i>;
            --entr-scl: unset<i>;
            --entr-rtt: unset<i>;
            --entr-mv-x: unset<i>;
            --entr-mv-y: unset<i>;
          `,
        },
      },
    },
  },
};

export default shilpConfig;
```
