# Transform Shape

Transform style for elements with 2D or 3D effects.

Usage:

- `@adjust shape-*;`
- `@adjust shape-2d;`
- `@adjust shape-3d;`

<Alert variant="info">

This page contains only the **property configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

<section>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  properties: {
    adjust: {
      shape: {
        property: "transform-style: <v><i>;",
        values: {
          "2d": "flat",
          "3d": "preserve-3d",
        },
      },
    },
  },
};

export default shilpConfig;
```
