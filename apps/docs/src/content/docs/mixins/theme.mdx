# @theme Mixin

Use `@theme` mixin for **dark/light** mode and **custom theme**.

Usage:

```css
.any-class {
	/* direct */
	@theme <variant> { ... }
}
```

<Alert variant="info">

This page contains only the **mixins configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  mixins: {
    theme: {
      variants: {
        /* class on parent, style on self */
        dark: ".dark &",
        light: ".light &",

        /* class on self, style on self */
        "dark-self": "&.dark",
        "light-self": "&.light",

        /* class on parent or self, style on self */
        // "dark-any": ".dark &, &.dark",
        // "light-any": ".light &, &.light",
      },
    },
  },
};

export default shilpConfig;
```
