# @motion Mixin

Use `@motion` mixin for respecting user motion preferences.

Usage:

```css
.any-class {
	/* direct */
	@motion <variant> { ... }
}
```

<Alert variant="info">

This page contains only the **mixins configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

---

<section>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  target: "react",

  mixins: {
    motion: {
      variants: {
        safe: "@media (prefers-reduced-motion: no-preference)",
        reduce: "@media (prefers-reduced-motion: reduce)",
      },
    },
  },
};

export default shilpConfig;
```
