# @siblings Mixin

Use `@siblings` mixin to style sibling elements.

Usage:

```css
.any-class {
	/* functions */
	@siblings <variant>("<arguments>") { ... }
}
```

<Alert variant="info">

This page contains only the **mixins configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

## Reference

<Alert>

Must see: [Youtube: CSS Siblings Selector](https://www.youtube.com/watch?v=rndMS4pEKP8&t=145s)

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  mixins: {
    siblings: {
      // https://www.youtube.com/watch?v=rndMS4pEKP8&t=145s

      /* functions */
      match: "& ~ <1>", // not nested, not from other parents, all following
      "match-closest": "& + <1>", // not nested, not from other parents, all direct following
    },
  },
};

export default shilpConfig;
```
