# @env Mixin

Use `@env` mixin for styling based on environment.

Usage:

```css
.any-class {
	/* direct */
	@env <variant> { ... }
}
```

<Alert variant="info">

This page contains only the **mixins configuration reference**. Complete documentation and usage
examples will be added progressively in future updates.

</Alert>

---

<section>

## Reference

```js title="shilp.config.js"
/** @type {import('shilpcss/types').ShilpConfig} */
const shilpConfig = {
  source: "react",

  mixins: {
    env: {
      variants: {
        // orientation
        tall: "@media (orientation: portrait)",
        wide: "@media (orientation: landscape)",

        // media type
        print: "@media print",
        screen: "@media screen",
      },
    },
  },
};

export default shilpConfig;
```
