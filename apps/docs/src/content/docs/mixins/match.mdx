# @match Mixin

Use `@match` mixin for complex selector patterns.

Usage:

```css
.any-class {
	/* functions */
	@match <variant>("<arguments>") { ... }
}
```

This page contains only the **mixins configuration reference**. Complete documentation and usage

/** @type {import('shilpcss/types').ShilpConfig} */
