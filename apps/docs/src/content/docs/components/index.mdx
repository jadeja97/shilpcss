# Components Overview

**Components** are predefined **CSS classes** provided by Shilp CSS.

Components are **statically defined and can also be generated dynamically** through the config
system.

To understand about components and it's processing, See:

- [components.css Default Styles](/docs/default-styles/components)
- [Components Config](/docs/config/components)

This section lists all built-in components shipped with Shilp CSS.
